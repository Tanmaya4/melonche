# melonche

Rebranded premium corporate website for **Meloche Windows & Doors** (previously Aura Architecture Studio).

## Features
- **Modern Corporate Landing Page**: Showcasing custom windows, premium entry/patio doors, and professional installation.
- **Teal & Navy Palette**: Rebranded styles using Meloche's signature colors (`#004057` and `#29858e`).
- **Interactive Products/Services Section**: Features dedicated tabs and lists for vinyl/wood windows, front doors, sliding patio doors, specialty configurations, and free assessments.
- **Project Case Studies**: Highlights local installation projects in Windsor and LaSalle, Ontario.
- **Fully Sanitized Links**: All broken template paths have been cleaned up and replaced with functional relative anchors.
- **Webflow Branding Removed**: The "Made in Webflow" badge is fully hidden via CSS overrides and dynamically destroyed using a runtime `MutationObserver` script.
- **Production Ready**: Bundles cleanly using Vite.
